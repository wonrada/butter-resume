## About

A hand-coded, no-framework personal site. Static HTML + CSS + vanilla JS only.

**Design highlights**
- Custom CSS cursor with `requestAnimationFrame` trail
- Scroll-triggered fade-in animations via `IntersectionObserver`
- Fully responsive — mobile, iPad, desktop (5 breakpoints)
- Collapsible experience timeline cards
- Phone mockup demo embedded in project case study
